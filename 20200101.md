# Kata
- [[Persistent-Bugger]](https://www.codewars.com/kata/persistent-bugger/javascript)
- 난이도 낮음 

# Solution
```javascript
function persistence(num) {
  let count = 0;
  while(num.toString().length > 1) {
    num = num.toString().split('').reduce((p,v)=>Number(v)*p,1); 
    count++;
  }
  return count;
}
```
